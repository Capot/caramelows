Caramelows Pixel

Description

Collection of 2000 pixel Caramelows. This collection is bio-inspired by my caramel dog Frida and her housemate Tieta.

Tieta was a street dog that we took in, and there will be some NFTs of her, Frida being the main star for symbolizing the most adorable breed in Brazil.

Mounting

This project was entirely made by me and entirely done in python, the best programming language :)

All the art was developed with the RGB concept to compose the images. So for that I made many many many 50 x 60 matrices.

I used a function to smooth the pixel points, an artifice that was also used in old game emulators.

suave = np.vstack([np.hstack([cv2.medianBlur(img, 3)]), ])

The randomization was done through a simple python function, which allows generating weights for each item of the elements:

self.oculos = rd.choices(la.oculos, weights=[8, 1, 1, 1, 1, 1, 1, 1, 1], k=1).

If the project is well accepted, I intend to make the code available in the future!!
